Einstellungen

Die Seite ermöglicht das Filtern der Quests, damit ihr die finden könnt, nach denen ihr gerade sucht.

Einreichung

Damit ihr eine Quest einreichen könnt, brauchen wir diese Informationen: 1. Name des Pokéstops 2. Die Belohnung der Quest 3. Den Quest-Typ 4. Wie oft die Quest ausgeführt werden muss.

Häufige Fragen

Warum ist die Seite leer?
- Jeden Tag um 0:00 verändern sich die Quests und deshalb wird die Seite zurückgesetzt. Seitdem hat noch niemand eine Quest eingereicht.

Warum werden nicht alle Quests aufgelistet?
- Es gibt aktuell keine Map, daher können nur die Quests angezeigt werden, die ein Spieler eingereicht hat.

Warum fragt die Seite nach meinem Standort?
- Der Standort wird benutzt, um die Liste der Stops nach deiner Distanz zu ihnen sortieren zu können. Die Standortdaten bleiben auf deinem Handy und werden nicht an den Server übermittelt.

Kann ich etwas ähnliches für meine Stadt erstellen?
- Ja! Du kannst den Code von Github herunterladen (siehe Informationen). Es sind ein wenig Kenntnisse von Node.js Socket.io und javascript nötig.

Ich kann den Pokéstop nicht finden?
- Die Daten wurden aus der Ingress Intel-Map importiert. Dabei wurden allerdings nur die Pokéstops in der des Zentrums von Aachen berücksichtigt. Du kannst dich bei uns über Discord melden, wenn die Pokéstops in deiner Nähe hinzugefügt werden sollen.

Was ist diese Seite?

Wir versuchen hier eine Liste aller an diesem Tag erhältlichen Quests in Aachen zu erstellen. Dafür ist die Mitarbeit der Community nötig.

Wie funktioniert das?

Es gibt aktuell keine verfügbare Map auf Bot-Basis, daher wurden alle Infos hier von Nutzern eingereicht.

Die Pokéstops wurden aus der Ingress Intel-Map exportiert, daher sind ein paar Stops dabei, die es gar nicht gibt. Außerdem wurden nur die Pokéstops in der Nähe des Zentrums von Aachen importiert. som är en sida där personer kan registrera Gym och Stop.

Die Seite basiert auf npm, node und "socket.io", ist also eine reine Web-App. "Socket.io" wird gerne in Kommunikations-Diensten verwendet, damit sind Informationen die von einem Nutzer eingereicht werden automatisch für alle Anderen ohne erneutes Laden der Seite sichtbar.

Gut zu wissen

Die Datenbank umfasst aktuell ca. 7500 Pokéstops im Stadtgebiet von Aachen, wobei manche dabei nur in Ingress vorhanden sind und keine echten Pokéstops sind. Falls Pokéstops fehlen, könnt ihr uns über Discord erreichen, damit wir sie hinzufügen.

Da alle Quests von Nutzern eingereicht werden, kann die Korrektheit der Informationen natürlich nicht garantiert werden, Trolle könnten theoretisch falsche Quests einreichen. Wir verlassen uns aber auf die Community, dass alle die Seite im guten Sinne benutzen.

GPS

Damit die Seite läuft wie gewünscht, müsst ihr den Zugriff auf euren Standort erlauben. Die Standort-Informationen bleiben dabei auf dem Gerät und werden nicht an der Server übermittelt.

Ziele des Projekts

Es soll so einfach wie möglich sein, neue Quests einzureichen.

Die Daten der Quests sollen in Zukunft in möglichst vielen Formaten angeboten werden(js, json, jsonp, lines m.m.), damit externe Entwickler die Daten für andere Zwecke nutzen können.

Wir (Das PokeKarte-Team) haben nur die Pokéstops in Aachen eingefügt und die Texte übersetzt, ursprünglich programmiert wurde die Seite von Mattias Lidbeck. Der gesamte Code lässt sich auf Github finden (https://github.com/teddykladdkak/quest-notifier-pogo) und kann von jedem verwendet werden. Hinweis von ihm: Beachtet dabei bitte, dass das Projekt noch jung und instabil ist, updatet euren Code regelmäßig, da ich fast täglich Änderungen vornehme..
